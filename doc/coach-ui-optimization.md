# 个人教练页面UI优化

**日期**: 2026-01-18  
**类型**: UI优化

## 优化内容

参考情感客厅页面的样式，优化了个人教练页面的AI消息展示：

### 新增特性

1. **微信风格聊天布局**
   - 用户消息靠右，AI消息靠左
   - 添加头像（用户显示"我"，教练显示"✨"）
   - 添加昵称标签（教练显示"✨ 情感教练"）

2. **消息气泡优化**
   - 用户消息：粉紫渐变背景，白色文字
   - 教练消息：白色背景，橙色边框
   - 圆角设计更柔和

3. **富文本支持**
   - 支持 Markdown 格式（加粗、斜体）
   - 标题自动高亮显示
   - 保持原有的思考过程折叠功能

### 技术细节

- 头像使用渐变色背景
- 消息气泡最大宽度70%
- 保留了思考过程的折叠展开功能
- 流式输出时同步更新样式

### 文件修改

- `templates/coach.html`: 更新CSS样式和JavaScript渲染逻辑

## 效果

个人教练页面现在与情感客厅保持一致的视觉风格，提升了用户体验的连贯性。
